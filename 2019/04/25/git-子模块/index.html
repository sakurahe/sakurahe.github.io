删除子模块

有添加，当然也会有删除。以删除 themes/aircloud 为例：

  • 使用 git rm --cached themes/aircloud 将themes/aircloud从版本控制中删除（本地仍保留），若不需要可以不带 --cached 进行完全删除
  • 使用 rm -rf .git/modules/themes/aircloud, 删除.git下的缓存模块，最后提交项目。

经过上面的删除后还可以进行添加子模块。
